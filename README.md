# SGR Anonymizer для TXT файлов

Публичный репозиторий с приложением для анонимизации диалогов в формате TXT по ФЗ-152.

## Описание

Приложение для анонимизации персональных данных в текстовых диалогах с использованием SGR методологии.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python test_random_5_fixed.py
```

## Структура

- `sgr_anonymizer/` - основной код анонимизатора
- `test_random_5_fixed.py` - скрипт для обработки TXT файлов
- `llm_client.py` - клиент для работы с LLM
